      社团活动

      在建设多元化课程体系的办学理念下，我校开设了丰富多彩的课外学生社团活动，

      编程社团旨在开阔学生的知识视野，培养学生编程兴趣，发掘其在程序设计方面的潜能与才华，提供展现自我的平台。

      学校举办趣味编程社团活动，通过Scratch趣味编程软件，为同学们搭建一个表达创意的平台，创造交互式故事、动画、游戏、音乐和艺术等作品。
